# E-commerce-Website
An interactive E-Commerce Website built with HTML, CSS, JavaScript, and Tailwind CSS. This project demonstrates product listing, cart management, wishlist functionality, and a responsive design suitable for desktop and mobile.
Responsive navigation bar with mobile menu toggle

Product listing fetched dynamically using fetch()

Add to Cart and Wishlist functionality (with localStorage support)

Cart summary and total calculation

Form handling for Sign Up and Order Checkout

Custom popup messages and hover effects

Modern styling with Tailwind CSS
