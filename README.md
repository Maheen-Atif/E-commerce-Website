# E-Commerce Website

This is a simple and beginner-friendly E-Commerce Website built using HTML, CSS, and JavaScript.  
The website allows users to sign up, sign in, browse products, add to cart, add to wishlist, and checkout easily with auto-filled user details.  
It also includes an Admin Dashboard for managing products (add/remove).

---

##  Features

## User Side
- 📝 **Sign Up & Sign In** — Create an account and log in to access the home page.  
- 🏠 **Home Page** — Browse available products.  
- 🛒 **Add to Cart** — Save items you want to purchase.  
- 💖 **Add to Wishlist** — Save your favorite products for later.  
- 🧾 **Checkout Page** — After adding products to the cart, proceed to checkout.  
- ✍️ **Auto-Fill Checkout Form** — Your name, email, and address are automatically filled.  
- ✅ **Order Confirmation** — Get a confirmation message once the order is placed.

## 🧰 Tech Stack
- HTML5 — Structure  
- Tailwind CSS — Styling  
- JavaScript  — Functionality  
- LocalStorage — For storing user data, cart, wishlist, and products
